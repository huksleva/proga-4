## 1. ВЫПОЛНЕННЫЕ ТРЕБОВАНИЯ

### 1.1. Основное задание

**Маршрут `/size2json`**  
- Принимает PNG изображения через `multipart/form-data`
- Возвращает JSON с размерами: `{"width": 1920, "height": 1080}`
- Проверка формата файла (только PNG)
- При неверном формате: `{"result": "invalid filetype"}`

**Маршрут `/login`**  
- Возвращает логин автора: `{"author": "1153307"}`

**JSON ответы**  
- Все данные возвращаются в формате JSON
- Правильные заголовки `Content-Type: application/json`

## 2. ВЫПОЛНЕННЫЕ УСЛОЖНЕНИЯ

### 2.1. Реализация фронтенда на Vue.js 3

**Использованные технологии:**
- **Backend:** Flask (Python)
- **Frontend:** Vue.js 3 (CDN)
- **CSS:** Кастомные стили с градиентами и анимациями

**Особенности реализации:**
- Реактивный интерфейс
- Компонентная структура
- Computed свойства для отображения имён файлов
- Асинхронные методы (async/await)

### 2.2. Асинхронная отправка данных

**Реализация:**
- Отправка файлов без перезагрузки страницы (Fetch API)
- Обработка ответов в реальном времени
- Индикация загрузки (`loading` state)
- Обработка ошибок

**Маршруты:**
- `POST /showPicture` — загрузка и отображение изображения
- `GET /lastUpload` — получение последнего загруженного файла
- `POST /clearUploads` — очистка папки uploads

### 2.3. Сохранение состояния

**Клиентская часть (localStorage):**
```javascript
localStorage.setItem('lastUpload', JSON.stringify(data));
```

**Серверная часть (JSON файл):**
```python
STATE_FILE = 'last_upload.json'
# Сохранение состояния между перезапусками сервера
```

**Функционал:**
- Приложение помнит последний загруженный файл
- Данные сохраняются после перезагрузки страницы
- Данные сохраняются после перезапуска сервера

### 2.4. Отображение изображений

**Возможности:**
- Предпросмотр выбранного файла (имя файла)
- Отображение загруженного изображения на странице
- Показ последнего загруженного файла
- Корректная обработка MIME-типов
- Автоматическое определение формата (PNG, JPEG, GIF, и т.д.)

### 3.2. Основные маршруты Flask

| Маршрут | Метод | Описание |
|---------|-------|----------|
| `/` | GET | Главная страница с формой |
| `/login` | GET | Возврат логина автора |
| `/size2json` | POST | Получение размеров PNG |
| `/showPicture` | POST | Загрузка и сохранение изображения |
| `/lastUpload` | GET | Получение последнего файла |
| `/clearUploads` | POST | Очистка папки uploads |
| `/uploads/<filename>` | GET | Доступ к загруженным файлам |

### 3.3. Валидация изображений

**Поддерживаемые форматы (PIL):**
- PNG, JPEG, JPG, GIF, BMP, WEBP, ICO
- TIFF, PSD, RAW, и другие (всего 40+ форматов)

**Проверка:**
```python
def is_allowed_image(file):
    img = Image.open(file.stream)
    img.verify()
    return img.format in PIL_FORMATS
```
