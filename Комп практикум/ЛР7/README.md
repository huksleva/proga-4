# S3 File Manager

Веб-сервис для управления файлами в S3-совместимом хранилище.

## Возможности

- ✅ Просмотр списка файлов в бакете
- ✅ Загрузка файлов
- ✅ Удаление файлов
- ✅ Получение presigned URL для доступа к файлам
- ✅ Скачивание и просмотр файлов
- ✅ Простой и удобный веб-интерфейс
