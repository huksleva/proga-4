# Лабораторная работа №14
## Веб-скрапинг сотрудников РГПУ им. А. И. Герцена
### Реализация с использованием BeautifulSoup

## Описание проекта

Данный проект реализует веб-скрапинг сотрудников РГПУ им. А. И. Герцена с сайта:

https://atlas.herzen.spb.ru/

В процессе работы программа:
1. Переходит по страницам преподавателей
2. Получает список сотрудников
3. Переходит на страницу профиля каждого сотрудника
4. Извлекает данные:
   - ФИО
   - ссылка на профиль
   - должность
   - факультет/институт
   - кафедра
   - учёная степень
   - почта
   - номер телефона
5. Сохраняет данные в CSV-файл

- Python 3

- BeautifulSoup4
- pandas
