# Создание проекта

Для создания проекта использовалась команда:

```bash
scrapy startproject herzen_scraper
```

После создания проекта был создан spider:

```bash
scrapy genspider teachers atlas.herzen.spb.ru
```

# Описание работы программы

## 1. Формирование списка страниц

В spider создаётся список страниц преподавателей:

```python
start_urls = [
    f"https://atlas.herzen.spb.ru/teachers?page={page}"
    for page in range(1, 55)
]
```

Таким образом spider автоматически проходит по всем страницам преподавателей.

---

# 2. Получение HTML-страниц

Основной метод обработки страниц:

```python
def parse(self, response):
```

Scrapy автоматически выполняет HTTP-запросы и передаёт HTML-ответ в объект response.

---

# 3. Поиск строк таблицы

Для поиска строк таблицы преподавателей используются CSS-селекторы:

```python
rows = response.css("tbody tr")
```

После этого выполняется проход по строкам таблицы.

---

# 4. Получение информации о преподавателях

Для каждой строки таблицы извлекаются:

* ФИО
* ссылка на профиль
* должность
* факультет
* кафедра
* учёная степень

Пример получения ФИО:

```python
fio = cells[0].xpath("string(.)").get("").strip()
```

Для корректного получения вложенного текста используется XPath:

```python
xpath("string(.)")
```

Данный способ позволяет получить весь текст элемента, включая вложенные HTML-теги.

---

# 5. Переход на страницу профиля

Для перехода на страницу профиля преподавателя используется:

```python
yield response.follow(
    profile_link,
    callback=self.parse_profile
)
```

Scrapy автоматически создаёт новый HTTP-запрос и вызывает метод обработки профиля.

---

# 6. Получение email и телефона

На странице профиля выполняется поиск элементов:

```python
h1_list = response.css(
    "h1.text-m::text"
).getall()
```

После этого выполняется поиск:

* email,
* номера телефона.

Пример поиска email:

```python
if '@' in string:
    email = string
```

Пример поиска телефона:

```python
if string.startswith(('+', '8')):
    phone_number = string
```

---

# 7. Формирование результата

Полученные данные возвращаются через yield:

```python
yield {
    "ФИО": fio,
    "Почта": email,
    "Номер_телефона": phone_number
}
```

Scrapy автоматически собирает полученные данные.

---

# 8. Создание CSV-файла

CSV-файл создаётся автоматически командой:

```bash
scrapy crawl teachers -o teachers_scrapy.csv
```

После выполнения программы создаётся файл:

```text
teachers_scrapy.csv
```

# Преимущества Scrapy

* Высокая производительность
* Поддержка асинхронности
* Автоматическая работа с HTTP-запросами
* Встроенная система очередей
* Поддержка middleware и pipelines
* Удобная архитектура для крупных проектов

---

# Недостатки Scrapy

* Более сложная структура проекта
* Более высокий порог входа
* Избыточен для небольших scraping-проектов

---

# Сравнение BeautifulSoup и Scrapy

| BeautifulSoup                   | Scrapy                        |
| ------------------------------- | ----------------------------- |
| Простая библиотека              | Полноценный framework         |
| Легко изучать                   | Более сложная архитектура     |
| Подходит для небольших проектов | Подходит для крупных проектов |
| Работа через requests           | Встроенная система запросов   |
| Более медленный                 | Более быстрый                 |
| Нет встроенной асинхронности    | Есть поддержка асинхронности  |
