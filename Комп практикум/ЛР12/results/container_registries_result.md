# Результат получения списка реестров контейнеров в Yandex Cloud

## Метод выполнения
Использован Yandex Cloud CLI (yc) для получения списка реестров контейнеров (Yandex Container Registry).

## Команды
```bash
yc container registry list
yc container registry list --folder-id b1gmddaiq52djbj6pgh2
```

## Результат

## Интерпретация
1. **Реестры отсутствуют** — в текущей папке Yandex Cloud не создано ни одного реестра контейнеров.
2. **Возможные причины**:
   - Реестры контейнеров ещё не созданы
   - Реестры находятся в другой папке или облаке
   - Недостаточно прав для просмотра реестров (маловероятно, так как команда выполнилась успешно)

## Рекомендации
1. **Создать новый реестр**:
   ```bash
   yc container registry create --name my-registry
   ```
2. **Проверить реестры в других папках**:
   ```bash
   yc resource-manager folder list
   yc container registry list --folder-id <другой-id-папки>
   ```
3. **Интеграция с лабораторной работой**:
   - Можно добавить функционал создания реестра через веб-интерфейс Flask-приложения
   - Реализовать API endpoint для управления реестрами контейнеров
